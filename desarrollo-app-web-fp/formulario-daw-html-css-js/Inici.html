Información

Copia barata de la famosa paágina de Skateboarding "Patin en Comun" del prestigioso Miquel Àngel Urbina.

Si navegáis por el menú superior podéis acceder a un test muy sencillo sobre el Skateboarding.

¡¡SKATE OR DIE!!

Instrucciones

Este questionario medirá el conocimiento que tienes sobre el skate.

Te encontrarás con diversas preguntas, unas solo tienen una respuesta posible a elegir entre varias; otras tendrás que responderlas escribiendo una única palabra y algunas tienen diversas respuestas correctas.

Debes responder a todas las preguntas para que se pueda corregir. Si pulsas sobre el logo, se te abrirá un la página de inicio.

¡Suerte!
